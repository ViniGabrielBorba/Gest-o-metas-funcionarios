# Frontend - Sistema de Gestão de Metas

Este é o frontend do sistema, desenvolvido com React e Tailwind CSS.

## Instalação

```bash
npm install
```

## Execução

```bash
npm start
```

O frontend estará disponível em `http://localhost:3000`

## Dependências

- React 18
- React Router DOM
- Tailwind CSS
- Recharts (gráficos)
- Axios (requisições HTTP)
- React Icons
